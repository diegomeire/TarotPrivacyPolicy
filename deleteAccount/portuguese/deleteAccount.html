Como apagar sua conta


1. Abra o aplicativo

2. Se você não tiver logado automaticamente, toque no botão de usuário para logar


3. Faça o login


4. Para remover a conta, toque novamente no botão de usuário na tela principal



5. Toque em "Remover Usuário"


6. Confirme as mensagens seguintes

ATENÇÃO!! ESTA AÇÃO É PERMANENTE E NÃO PODE SER DESFEITA. TODOS OS DADOS DO USUÁRIO SERÃO REMOVIDOS
